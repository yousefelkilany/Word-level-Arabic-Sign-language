# visualize_model_performance.py

#source #modelling #visualization

Generates plots for training metrics (Loss, Accuracy) and confusion matrices.

## Output
- Saves plots to `logs/` or `outputs/` directory.
- Used to verify model convergence and analyze error patterns.

## Usage
```bash
make visualize_metrics
```
